Ele identifica através de um modo ilustrativo(imagem) em qual turno(manhã, tarde e noite) está,  a base é através da hora do sistema.Este é o meu primeiro projeto que subo nesta plataforma, sendo assim um start para conseguir meu primeiro emprego como Dev.
